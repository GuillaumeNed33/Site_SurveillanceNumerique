Projet du site pour le PPP :

Notre site est composé de 9 pages HTML et de ? styles CSS.

Notre site utilise :

  • HTML 5
  • CSS 3

L'utilisation de HTML 5,nous a permis d'utiliser de nouvelles balises telles que :

  • header
  • footer
  • section
  • article
  • figure
  • figcaption

L'utilisation de CSS 3, nous a permis d'utiliser différents effets visuels comme:

  • border radius
  • box-shadow

Notre site n'est pas responsive, l'écran le plus favorable à son utilisation est un écran moyen d'ordinateur. Testé sous Mozilla Firefox et Chromium


Ce site a été réalisé à l'occasion d'un projet étudiant au département informatique de l'IUT de Bordeaux.
